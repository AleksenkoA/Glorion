# Glorion Landing Page

Лендинг-страница для Glorion Casino на итальянском языке.

## Структура проекта

```
glorion-it/
├── index.html          # Основная HTML страница
├── assets/
│   ├── css/
│   │   └── style.css   # Стили страницы
│   ├── js/
│   │   └── main.js     # JavaScript функциональность
│   └── images/         # Изображения (логотип, favicon)
└── README.md
```

## Особенности

- Полностью адаптивный дизайн
- Все тексты на итальянском языке (из ТЗ)
- Современный дизайн с анимациями
- Оптимизирован для мобильных устройств
- Секции включают:
  - Основные данные и сильные стороны
  - Методы оплаты
  - Игровой опыт для разных профилей
  - Каталог игр и провайдеры
  - Бонусы и условия
  - Безопасность и поддержка
  - Спортивные ставки
  - Миссии и награды
  - FAQ

## Использование

Откройте `index.html` в браузере для просмотра.

## Необходимые файлы

Добавьте следующие файлы в `assets/images/`:
- `logo.png` - логотип Glorion
- `favicon.svg` - иконка сайта

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript
